GLOBAL AZURE BOOTCAMP 2017

Plug-se na Nuvem Pública da Microsoft

Inscreva-se
vagas limitadas!

O Global Azure BootCamp, é um evento gratuito realizado anualmente em escala global, para que o conhecimento sobre Microsoft Azure seja disseminado entre a comunidade, incentivando a sua utilização e adoção.

Público Alvo

Desenvolvedores, arquitetos e it-pros com interesse em conhecer e aprender mais sobre a plataforma de Cloud Computing da Microsoft.

Networking

Aproveite para conhecer pessoas, fazer crescer sua rede de contatos que pode lhe proporcionar valor, desde a troca de valiosas informações sobre soluções na Nuvem, ou simplesmente para um bom papo sobre tecnologia.

Plug-se a Nuvem

Um dia de imersão completa que vai te deixar apto (pro) a desenvolver aplicações e/ou criar sua infraestrutura utilizando recursos de Cloud Computing no Microsoft Azure.

Por dentro dos recursos de infraestrutura do Microsoft Azure.

09:40 - 10:30

Infraestrutura no Azure

Está querendo saber um pouco sobre os recursos de infraestrutura disponiveis no Microsoft Azure? Então venha conosco! Vamos apresentar uma visão geral sobre as máquinas virtuais Azure e como implementar máquinas virtuais Windows ou Linux, Azure Storage, Azure Active Directory. Falaremos também sobre oportunidades de negócios e o mercado de trabalho.

Azure SQL Database

14:30 - 15:10

Azure SQL

O Azure SQL Database é a plataforma de banco de dados da Microsoft no Azure. Este é um produto que sempre estará atualizado, seguro e disponível. Então, venha descobrir como funciona o SQL Server na nuvem da Microsoft e entenda como economizar em hardware, licenciamento e depreciação.

Azure: Code & Deploy

15:20 - 16:00

Azure Code & Deploy

Nesta sessão iremos demonstrar na prática como implementar e desenvolver suas aplicações no Microsoft Azure de forma ágil e surpreendente, vamos decmosntrar como configurar o Visual Studio para o Deploy, além de algumas funcionalidades mais robustas como multiplas instâncias e load balance.
